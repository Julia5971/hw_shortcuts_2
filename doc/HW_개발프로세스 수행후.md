\# 단축키 학습 센터

단축키 학습을 위한 웹 기반 플랫폼입니다. 다양한 프로그램의 단축키를 쉽게 학습하고 관리할 수 있습니다.

\#\# 주요 기능

\- 다양한 프로그램의 단축키 학습  
\- 실시간 검색 기능  
\- 학습 진행도 관리  
\- 즐겨찾기 기능  
\- 반응형 디자인

\#\# 기술 스택

\- HTML5  
\- CSS3  
\- JavaScript (ES6+)  
\- LocalStorage API

\#\# 설치 및 실행

1\. 저장소 클론  
\`\`\`bash  
git clone https://github.com/yourusername/shortcut-learning-center.git  
\`\`\`

2\. 프로젝트 디렉토리로 이동  
\`\`\`bash  
cd shortcut-learning-center  
\`\`\`

3\. 웹 서버 실행  
\- VS Code의 Live Server 확장 프로그램 사용  
\- 또는 로컬 웹 서버 사용
